Cyber Adventure

Một tựa game nhập vai phiêu lưu với đồ họa 3D tuyệt đẹp và cốt truyện hấp dẫn. Khám phá thế giới rộng lớn với nhiều bí ẩn chờ đợi bạn khám phá.

Tính năng nổi bật:

  • Đồ họa 3D chân thực, sống động
  • Hệ thống nhân vật phong phú với nhiều kỹ năng đặc biệt
  • Thế giới mở rộng lớn với hàng trăm nhiệm vụ
  • Chế độ PvP và PvE đa dạng
  • Cộng đồng game thủ đông đảo

Lưu ý: Game yêu cầu hệ thống cấu hình tối thiểu Windows 10, RAM 8GB, card đồ họa 2GB.
